# 🏦 大富翁 · 金库

多人在线大富翁（Monopoly）辅助计钱工具。基于 Socket.IO 实时同步，移动端优先设计，让你的桌面桌游资金管理不再混乱。

## ✨ 功能

- **💰 资金操作** — 加钱（银行收入）、减钱（银行支出）、玩家间转账，三操作覆盖全部场景
- **⚡ 快捷事件** — 自定义收入/支出事件（如"过路费 +200"、"交税 −500"），一键触发，全房间同步
- **👥 多人房间** — 4位房间码加入，最多8人，Token 断线重连，刷新页面不丢失
- **📋 交易记录** — 每条资金变动实时记录，折叠式查看，上限200条防卡顿
- **🎨 高端深色UI** — 金色主题、资金大字展示、弹跳动画，手机触屏优化

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/tianshiemo7/monopoly-money.git
cd monopoly-money

# 安装依赖
npm install

# 启动服务器
npm start
```

服务器启动后：
- **本机**访问 `http://localhost:3000`
- **手机**连同一 WiFi，访问 `http://<电脑IP>:3000`（启动日志会显示）

## 🎮 使用流程

1. **一人创建房间** → 设定初始资金（默认 ¥1,500），获得4位房间码
2. **其他人加入** → 输入房间码和昵称
3. **开始管理资金**：
   - 输入金额点击 `+` `−` 手动加减
   - 创建快捷事件，一键触发常见操作
   - 点击其他玩家旁的 `💸` 进行转账
4. **所有操作实时同步**，交易记录可追溯

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 运行时 | Node.js |
| HTTP 框架 | Express |
| 实时通信 | Socket.IO (WebSocket + Polling fallback) |
| 前端 | 原生 HTML/CSS/JS，零依赖 |
| 状态管理 | 服务端权威 |

## ⚙️ 配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 端口 | 3000 | 可修改 server.js 中 PORT 变量 |
| 初始资金 | ¥1,500 | 创建房间时可自定义 |
| 最大玩家 | 8 | 房间容量 |
| 日志上限 | 200 条 | 超出自动裁剪 |
| 房间过期 | 2 小时 | 无人活动自动清理 |
| 离线宽限 | 30 秒 | 断线后保留位置 |

## 🔒 安全

- 输入消毒：剥离 HTML 标签、引号、换行符
- 输出转义：所有用户内容渲染前 XSS 转义
- Payload 限制：Socket.IO 最大 100KB
- 金额上限：单次操作 ≤ ¥99,999,999
- 事件上限：每房间最多 50 个自定义事件
